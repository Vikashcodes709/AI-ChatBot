# 🤖 Vikash AI ChatBot

A simple AI ChatBot built using Python, Streamlit, and Google Gemini API.

## 🚀 Features

- AI Chat using Gemini
- Clean Streamlit UI
- Fast Responses
- Easy to Use

## 🛠️ Technologies Used

- Python
- Streamlit
- Google Gemini API

## ▶️ Run Project

Install requirements:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python -m streamlit run app.py
```
